#git branching

1. git branch 
=>list all the branches and highlights the current/active branch

2. git branch <branch_name>
=>create a new branch

3. git checkout <branch_name>
=> jumps to the branch name provided

4. git checkout -b <branch_name>

5. git merge branch1
=> merge main branch and local branch 

6. git push
=> push local to main github .
